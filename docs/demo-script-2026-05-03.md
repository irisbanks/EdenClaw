# EDENCLAW 3단계 시연 스크립트

작성일: 2026-05-03

## 목표

EDENCLAW가 “중고거래 소비자와 판매자를 위한 AI 매매 의사결정 엔진”이라는 점을 4분 내에 설득한다.

핵심 메시지:

- 한국 중고거래 맥락에 맞춘 가격 분석과 협상 전략
- 글로벌 가격 폴백을 통한 차익거래 검증
- GPT, Gemini, Claude, Edenclaw AI를 동시에 비교하는 4-way AI 라우터
- 자체 vLLM 기반 Edenclaw AI는 외부 API 비용 없이 동작 가능

## 1. 오프닝 (30초)

### 발표 멘트

“중고거래에서 가장 어려운 건 세 가지입니다. 이 가격이 맞는지, 어떻게 협상해야 하는지, 그리고 해외 가격과 비교했을 때 진짜 기회가 있는지입니다.

EDENCLAW는 이 세 가지를 AI가 즉시 검증하는 매매 비교 시스템입니다. 사용자는 상품명과 가격만 입력하면, ExpertTrader가 국내 시세, 내부 거래 DB, 글로벌 가격 폴백을 보고 협상 전략을 만들고, 필요하면 GPT, Gemini, Claude, 자체 Edenclaw AI까지 동시에 비교합니다.

오늘은 협상, 차익거래 검증, 4-way AI 비교를 순서대로 보여드리겠습니다.”

### 화면

- `http://localhost:3000` 또는 데모 페이지
- Expert 응답 API와 AI 비교 대시보드 준비
- 네트워크/외부 API 실패 시에도 fallback 결과가 나오는 점을 미리 언급

## 2. 라이브 시연 (3분)

## 2-1. 협상 전략 시연 (60초)

### 입력 예시

```json
{
  "intent": "negotiate",
  "userPrice": 600000,
  "itemDescription": "다이슨 V15 무선청소기"
}
```

### 진행 멘트

“첫 번째는 협상입니다. 사용자가 다이슨 V15를 60만원에 거래하려고 합니다. EDENCLAW는 먼저 상품명을 기준으로 내부 DB와 외부 가격 소스를 확인합니다. 정확 매칭이 있을 때만 내부 DB 평균을 사용하고, 매칭이 약하면 무리하게 시세를 만들지 않습니다.”

### 보여줄 포인트

- 입력가
- 시세 또는 데이터 부족 처리
- 최저 양보가
- 예상 거래 기간
- 협상 성공률
- 평균 최종 거래가

### 강조 멘트

“이전 버전에서는 내부 DB fallback이 너무 넓어서 모든 상품이 같은 평균가로 잡히는 문제가 있었습니다. 지금은 상품명 핵심 토큰이 맞지 않으면 ‘정확 매칭 없음’으로 처리하고, 잘못된 평균가를 보여주지 않습니다.”

## 2-2. 차익거래 검증 시연 (60초)

### 입력 예시

```json
{
  "message": "아이폰 15 Pro 256GB 100달러 예산 차익거래",
  "scenario": "arbitrage",
  "budget": 100
}
```

### 진행 멘트

“두 번째는 차익거래입니다. 여기서 중요한 건 무조건 큰 숫자를 보여주는 게 아니라, 실제 예산으로 실행 가능한지 검증하는 것입니다. 100달러 예산으로 아이폰 15 Pro 차익거래를 찾으면, 시스템은 고가 상품 후보를 잘라내고 안전한 기회가 없다고 판단합니다.”

### 보여줄 포인트

- 후보 0건 또는 `No viable opportunity`
- 예산 초과 후보 필터링
- `net_profit_usd`를 상품 가격처럼 표시하지 않음
- 아이폰 검색에 소니 렌즈 같은 오매칭이 붙지 않음

### 강조 멘트

“차익거래 데모에서 가장 위험한 건 그럴듯한 거짓 수익입니다. EDENCLAW는 이제 실제 구매가, 필요 자본, 순이익을 분리하고, 예산을 넘으면 실격 처리합니다. 그래서 보여주기 좋은 숫자보다 실행 가능한 판단을 우선합니다.”

## 2-3. 4-way AI 비교 시연 (60초)

### 입력 예시

```json
{
  "message": "iPhone 15 Pro Max 256GB 사고 싶어",
  "scenario": "buy"
}
```

### 진행 멘트

“마지막은 4-way AI 비교입니다. 같은 요청을 GPT, Gemini, Claude, Edenclaw AI에 동시에 보내고 결과를 한 화면에서 비교합니다. 가격, 플랫폼, 응답 속도, 리스크, 추천 논리성을 기준으로 종합 점수를 계산합니다.”

### 보여줄 포인트

- GPT-5.5 또는 OpenAI fallback
- Gemini
- Claude
- Edenclaw AI, 자체 vLLM Qwen2.5-72B
- 우승 AI와 점수표
- HTML 리포트
- 비용 분석

### 강조 멘트

“이 비교는 Edenclaw가 무조건 1등이 되도록 만든 구조가 아닙니다. 속도와 비용 가중치는 낮추고, 시장 시세 근접성, 추천 논리성, 리스크 회피 능력을 더 크게 반영하도록 수정했습니다.”

## 3. 차별화 포인트

## 한국 특화

- 원화 가격 기준 협상 전략
- 국내 중고거래 스타일에 맞는 최저 양보가, 평균 최종 거래가, 예상 거래 기간 제공
- 네이버 쇼핑, 내부 거래 DB, 국내 직거래 상황을 우선 고려

## 글로벌 6개 폴백

데모 메시지:

“한국 시세만 보면 놓치는 기회가 있고, 해외 시세만 보면 실행 불가능한 기회가 많습니다. EDENCLAW는 국내와 글로벌 데이터를 함께 보고, 실패 시에도 단계별 fallback을 탑니다.”

폴백 구조:

- Naver Shopping
- Amazon
- Newegg
- AliExpress
- Wish
- 11번가 및 내부 DB

## 0원 비용 구조

데모 메시지:

“외부 API는 강력하지만 매 호출마다 비용이 발생합니다. EDENCLAW 자체 AI는 vLLM 기반 Qwen2.5-72B를 로컬 GPU에서 돌리기 때문에, 토큰 과금 없이 운영할 수 있습니다. 대량 비교나 반복 시뮬레이션에서 비용 우위가 생깁니다.”

강조 포인트:

- 외부 API: 고품질 추론과 비교 기준
- Edenclaw AI: 자체 서버, 빠른 반복, 토큰 비용 0원
- Redis/DB 캐시와 결합하면 운영비를 더 줄일 수 있음

## 정확성 방어

- 내부 DB fallback은 상품명 핵심 토큰 매칭이 있을 때만 사용
- Naver scope pending 또는 Amazon quota 초과 시 경고를 명시
- 데이터 부족이면 과감히 “추천 불가” 또는 낮은 신뢰도로 표시
- 차익거래는 예산 초과와 상품 오매칭을 실격 처리

## 4. 사업화 및 예상 Q&A

## 사업화 포인트

### 소비자용

- “살까 말까” 가격 판단
- 협상 메시지 자동 생성
- 사기 위험과 과도한 가격 탐지

### 판매자용

- 적정 판매가 산정
- 판매글 초안 생성
- 해외/국내 가격 비교 기반 판매 채널 추천

### B2B/API

- 리셀러용 차익거래 검증 API
- 중고거래 플랫폼용 AI 가격 검수
- 대량 상품 등록 가격 추천
- 글로벌 마켓 가격 차이 모니터링

## 예상 Q&A

### Q1. Naver API가 실패하면 어떻게 하나요?

A. 실패를 숨기지 않고 warning으로 노출합니다. 동시에 내부 DB와 글로벌 소스 fallback을 사용합니다. 다만 정확 매칭이 없으면 임의 평균가를 만들지 않고 낮은 신뢰도 또는 데이터 부족으로 처리합니다.

### Q2. AI가 틀린 가격을 추천하면 어떻게 방지하나요?

A. 가격 소스와 신뢰도를 분리합니다. 내부 DB는 핵심 키워드 매칭률을 통과해야만 사용하고, 차익거래는 예산 초과 후보를 제거합니다. 4-way 비교에서도 단순 속도보다 시장 근접성과 추천 논리성을 더 크게 평가합니다.

### Q3. 차익거래 수익률이 너무 커 보이면 믿어도 되나요?

A. 바로 믿으면 안 됩니다. EDENCLAW는 구매가, 필요 자본, 판매가, 수수료, 순이익을 분리해 보여주고, 상품명 오매칭이나 예산 초과가 있으면 실격 처리합니다. 데모에서도 “가짜 수익”보다 “실행 가능한 기회”를 우선합니다.

### Q4. 왜 4개의 AI를 동시에 쓰나요?

A. 단일 AI는 편향이나 환각이 생길 수 있습니다. 여러 AI가 같은 요청에 대해 어떤 가격, 플랫폼, 리스크 판단을 하는지 비교하면 신뢰도를 높일 수 있습니다. 최종 추천은 점수표와 근거를 함께 보여줍니다.

### Q5. Edenclaw AI가 외부 AI보다 좋은 점은 무엇인가요?

A. 외부 AI는 일반 추론이 강하고, Edenclaw AI는 로컬 상품 DB와 한국 거래 맥락에 가깝습니다. 특히 자체 vLLM 서버라 반복 실행 비용이 낮습니다. 그래서 대량 매물 분석이나 상시 모니터링에 적합합니다.

### Q6. 지금 바로 시연 가능한 범위는 어디까지인가요?

A. 협상 전략, 안전한 차익거래 검증, 4-way AI 비교는 시연 가능합니다. 다만 실시간 외부 가격 API scope나 quota 상태에 따라 일부 결과는 fallback으로 표시될 수 있습니다. 이 경우에도 시스템은 경고와 신뢰도 하향을 명확히 보여줍니다.

## 마무리 멘트

“EDENCLAW의 핵심은 AI가 대신 거래를 단정하는 것이 아니라, 사용자가 더 안전하고 빠르게 판단할 수 있도록 가격, 협상, 리스크, 글로벌 비교를 한 번에 정리하는 것입니다. 한국 중고거래에 맞춘 실행형 AI 매매 엔진으로 확장할 수 있습니다.”
